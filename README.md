# msmith8846.github.io

Personal portfolio for **Mario Smith** — Computer Information Systems student at Niagara University.

Live site: https://msmith8846.github.io

## Stack
- Static HTML / CSS / vanilla JavaScript
- HTML Canvas for background particles and the Slapshot game
- Hosted on GitHub Pages from `main`

## Pages
- `index.html` — landing, about, experience, projects, skills, contact
- `blackjack.html` — a browser rebuild of the original Java Blackjack project
- `slapshot.html` — a canvas-based hockey mini-game

## Dev
No build step. Open `index.html` in a browser, or serve locally:
```
python -m http.server 8000
```
